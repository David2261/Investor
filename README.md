## Getting Started
- Virtual environment
	- pip install virtualenv
- Activate
	- venv\Scripts\activate
- Install tools
	- pip install -r requirements
- Run django server
	1) python manage.py makemigrations
	2) python manage.py migrate
	3) python manage.py runserver
- Run Node server
```bash
cd hello/apps/frontend
npm run dev
```
